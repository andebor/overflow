Codebase for [overflow.no](https://overflow.no)

## Backend

Set up the Python environment and install dependencies

#### Install

```
cd backend
apt install python3 python3-pip
make env
source env/bin/activate
make install
```

#### Test

```
make test
```

#### Run

```
make run
```
